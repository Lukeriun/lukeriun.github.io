Java

Java é uma linguagem de programação que foi desenvolvida na decada de 90 por um grupo de pessoas numa empresa chamada Sun Mycroworks, que atualmente pertence à empresa Oracle, com o nome Project Green. Esse projeto na época em que foi criado não deu certo, mas deu origem a linguagem Java

Programas famosos que usam Java

Uber, LinkedIn, Spotify, Acrobat Reader, Netflix, Amazon, Google Docs, Nasa World Wind, Eclipse e Minecraft
