A historia das linguagens de programação

Por: Alexandre Luca, Victor Hugo, Nathan, Arthur e Pietro

Sobre o projeto

Nosso projeto tem como objetivo trazer algumas curiosidades sobre as linguagens de programação. Quando foram criadas, para que servem e outras coisas. Vamos tentar uma abordagem básica, para ser de facil entendimento para um publico geral.

O que é programação?

Programação nada mais é do que processo de criação e manutenção de um programa de computador. Esse programa é escrito em uma linguagem de programação, embora seja possível escrevêlo, com mais dificuldade, em uma linguagem de máquina. Diferentes partes de um programa podem ser escritos em linguagens de programação diferentes.
